## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## DATABASE MIGRATION

```bash
# unit tests
$ yarn migrate:up

# e2e tests
$ yarn migrate:down

# test coverage
$ yarn migrate:create name # where name is the name of the migration
```
